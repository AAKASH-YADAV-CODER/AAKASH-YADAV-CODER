# Aakash Yadav

Backend-focused full-stack developer building production-grade systems with NestJS, TypeScript, Redis, and modern AI workflows.

I care about clean architecture, performance, reliability, and developer experience. I enjoy turning product ideas into maintainable, scalable software.

## What I Build

- Modular backend services with NestJS and TypeScript
- High-performance APIs with Redis caching and optimized data access
- Async workflows with queues and event-driven patterns
- AI-powered product features using retrieval, orchestration, and guardrails

## Core Expertise

**Backend Engineering**
- NestJS, Node.js, TypeScript, Express
- REST API design, auth, validation, rate limiting
- Clean architecture and domain-driven service boundaries

**Data and Performance**
- Redis (cache strategy, TTL design, cache invalidation)
- MongoDB and MySQL
- Query optimization and API latency reduction

**AI Engineering**
- LLM integration and tool-calling workflows
- RAG fundamentals: chunking, embeddings, retrieval, and context assembly
- Basic evaluation thinking: relevance, faithfulness, latency, and cost
- Guardrails, fallback paths, and reliable AI UX patterns

**Quality and Delivery**
- Jest, Postman, GitHub-based workflows
- CI/CD mindset, code review discipline, and testing-first habits
- Docker and Linux-based development environments

## Engineering Principles

- Build for clarity first, then optimize where data shows bottlenecks
- Keep services observable and failure-tolerant
- Prefer simple, testable modules over tightly coupled abstractions
- Treat AI features as production systems, not demos

## Current Focus

- Designing scalable NestJS APIs and service boundaries
- Applying Redis for caching, background processing, and throughput gains
- Building robust AI features with measurable quality and safety
